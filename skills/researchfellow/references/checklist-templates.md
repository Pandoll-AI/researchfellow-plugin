# Reporting Checklist Templates

## STROBE Checklist (Observational Studies)

Use for cohort, case-control, and cross-sectional studies.

| # | Item | Section | Description |
|---|------|---------|-------------|
| 1a | Title | Title | Study design in title |
| 1b | Abstract | Abstract | Structured summary (background, methods, results, conclusions) |
| 2 | Background | Introduction | Scientific background and rationale |
| 3 | Objectives | Introduction | Specific objectives, including pre-specified hypotheses |
| 4 | Study design | Methods | Key elements of study design early in paper |
| 5 | Setting | Methods | Setting, locations, relevant dates (enrollment, exposure, follow-up, data collection) |
| 6a | Participants | Methods | Eligibility criteria, sources/methods of selection |
| 6b | Participants | Methods | For matched studies: matching criteria and number of matched per case |
| 7 | Variables | Methods | All outcomes, exposures, predictors, confounders, effect modifiers clearly defined |
| 8 | Data sources | Methods | Sources of data and details of assessment methods |
| 9 | Bias | Methods | Efforts to address potential bias |
| 10 | Study size | Methods | How study size was arrived at |
| 11 | Quantitative variables | Methods | How quantitative variables were handled, grouping choices |
| 12a | Statistical methods | Methods | All statistical methods including confounders |
| 12b | Statistical methods | Methods | Subgroups and interactions |
| 12c | Statistical methods | Methods | Missing data handling |
| 12d | Statistical methods | Methods | Loss to follow-up |
| 12e | Statistical methods | Methods | Sensitivity analyses |
| 13a | Participants | Results | Numbers at each stage (flow diagram) |
| 13b | Participants | Results | Reasons for non-participation |
| 13c | Participants | Results | Consider flow diagram |
| 14a | Descriptive data | Results | Characteristics of participants and exposures |
| 14b | Descriptive data | Results | Number with missing data per variable |
| 15 | Outcome data | Results | Number of outcome events or summary measures |
| 16a | Main results | Results | Unadjusted and adjusted estimates with CI and p-value |
| 16b | Main results | Results | Category boundaries for continuous variables |
| 16c | Main results | Results | Meaningful translations of relative measures |
| 17 | Other analyses | Results | Other analyses done (subgroup, interaction, sensitivity) |
| 18 | Key results | Discussion | Summary of key results with reference to objectives |
| 19 | Limitations | Discussion | Limitations including potential bias |
| 20 | Interpretation | Discussion | Cautious interpretation considering objectives, limitations, multiplicity |
| 21 | Generalizability | Discussion | Generalizability (external validity) |
| 22 | Funding | Other | Funding and role of funders |

## RECORD Checklist (Routinely Collected Data)

Extension of STROBE for studies using routinely collected health data (EHR, claims, registries).

Additional items beyond STROBE:

| # | Item | Section | Description |
|---|------|---------|-------------|
| R1 | Data source | Methods | Type of data (EHR, claims, registry) and purpose of collection |
| R2 | Data cleaning | Methods | Methods of data cleaning |
| R3 | Linkage | Methods | Record linkage methods, if applicable |
| R4 | Study period | Methods | Start and end dates of study period |
| R5 | Codes | Methods | Codes used to identify exposures, outcomes, covariates (e.g., ICD, ATC) |
| R6 | Code validation | Methods | Validation of codes used |
| R7 | Data access | Methods | Data access and cleaning methods (researchers vs analysts) |
| R8 | Sensitivity of codes | Results | Sensitivity analysis for code definitions |

## TRIPOD Checklist (Prediction Models)

For prediction/prognostic model development or validation studies.

| # | Item | Section |
|---|------|---------|
| 1 | Title | Identify as development/validation of multivariable prediction model |
| 2 | Abstract | Structured abstract |
| 3a | Background | Medical context and rationale |
| 3b | Objectives | Study objectives |
| 4a | Source of data | Design (cohort, RCT, etc.) |
| 4b | Data collection | How and when |
| 5a | Participants | Eligibility |
| 5b | Treatment | Any treatments received |
| 6a | Outcome | Clearly define outcome |
| 6b | Assessment | Blind to predictors? |
| 7a | Predictors | All predictors clearly defined |
| 7b | Assessment | Blind to outcome? |
| 8 | Sample size | How determined |
| 9 | Missing data | Handling approach |
| 10a | Model | Specify type |
| 10b | Predictors | Selection method |
| 10d | Validation | Internal or external |
| 11 | Risk groups | How created |
| 12 | Development vs validation | Report separately |
| 13a | Flow | Participant flow diagram |
| 14a | Characteristics | Demographics and predictors |
| 15a | Model development | Show full model |
| 15b | Performance | Overall and subgroups |
| 16 | Performance | Discrimination and calibration |
| 17 | Implications | Clinical use |
| 18 | Limitations | Study limitations |
| 19a | Interpretation | Results in context |
| 20 | Supplementary | Model specification for calculation |
